# dbt-exp

## Prerequisites

Tested with:

- Docker 20.10.22
- Python: 3.9.4

## How to run


#### Setup venv

```bash
python3 -m venv venv/
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

#### Run

```bash
source venv/bin/activate
python app.py
```
